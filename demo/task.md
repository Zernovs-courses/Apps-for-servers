# Демонстрационный вариант задания к зачету по дисциплине "Разработка программных модулей и приложений для серверов"

Напишите приложение FastAPI для получения, хранения и обработки данных о фильмах.

Для выполнения работы используйте файл `main.py`.

Каждая запись должна включать в себя следующие поля:
* `id`
* `title` - название фильма
* `year` - год премьеры
* `country` - страна
* `genre` - жанр

Способ хранения данных (словарь, текстовый файл или база данных SQL) выбрать на свое усмотрение.

API должен включать следующие endpoint:
* `/films` - получение списка фильмов
* `/films/{id}` - получение информации о фильме по его id
  * В случае отсутствия фильма возвращается `404 "Film not found"`
* `/films` - добавление нового фильма (используя тело запроса)
  * `id` - поле должно быть заполнено автоматически
  * В случае успешного добавления возвращается добавленный объект
* `/films/{id}` - обновление информации о фильме по его id (используя тело запроса)
  * В случае отсутствия фильма возвращается `404 "Film not found"`
  * В случае успешного обновления возвращается обновленный объект
* `/films/{id}` - удаление информации о фильме по его `id`
  * В случае отсутствия фильма возвращается `404 "Film not found"`
  * В случае успешного удаления возвращается `{"message": "Film deleted"}`
* `/films/filter` - фильтрация списка фильмов по заданным параметрам
  * В случае отсутствия фильмов, удовлетворяющих условиям, возвращается пустой список
  * В случае успешного фильтрации возвращается список фильмов
  * Параметры фильтрации: `year, genre, country, title`
  * Параметры фильтрации могут быть заданы в виде списка и словаря, например:
    * `/films/filter?year=2010`
    * `/films/filter?year=2010,2020`
    * `/films/filter?year={"from": 2010, "to": 2020}`
  * Фильтрация по `title` может быть производится по подстроке, например:
    * `/films/filter?title=The`
    * `/films/filter?title=the`
    * `/films/filter?title=The Matrix`
